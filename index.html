2. Resources
Topic Field Level Relevant Standard Definition Response Options Controlled Vocabulary Notes
Resources Type of resource Paper ClinicalTrials.gov The type of resource being shared. Text-CV Published data, code, consent forms, etc. Used to identify shared data/code documentation for replication.
Resources URL of resource Study ClinicalTrials.gov URL to access the resource. Open text — Permanent URLs like DOIs preferred.
Resources Registration ID Study ClinicalTrials.gov, AEA Registry Unique identifier from a registry. Open text — Used to cross-reference external trial registries.
Resources Registry URL Study AEA Registry Permanent URL or DOI to access the registration. Open text — Used to verify pre-registration details.
Resources Review board name Study ClinicalTrials.gov The name of the ethics review board. Open text — Used to assess ethical oversight.
Resources Review number Study ClinicalTrials.gov ID assigned by the review board. Open text — May be omitted if protocol not approved.

Topic Field Level Relevant Standard Definition Response Options Controlled Vocabulary Notes

4. Topics and Objectives
Topic Field Level Relevant Standard Definition Response Options Controlled Vocabulary Notes

5. Sampling
Topic Field Level Relevant Standard Definition Response Options Controlled Vocabulary Notes
Sampling Country Treatment effect DDI 2.5 Country or countries where the study was implemented. Text-CV ISO country codes Add "other" and "did not report" options.
Sampling Sub-national location Treatment effect DDI 2.5 Geographic location sampled within the country. Open text — Examples: villages in Busia, cities in Indonesia.
Sampling Unit of randomization Arm AEA Registry, ClinicalTrials.gov Unit at which randomization was done and treatment assigned. Open text and Text-CV From Cavanagh et al. (2023) Use controlled vocab from Appendix B and refine in pilot.
Sampling Unit of analysis Treatment effect ClinicalTrials.gov Unit at which treatment effect was measured. Open text and Text-CV From Cavanagh et al. (2023) Compare to unit of randomization to assess mismatch.
Sampling Eligibility of experimental population Study DDI 2.5, ClinicalTrials.gov Criteria for inclusion/exclusion in the experimental sample. Open text — Record for both unit of randomization and analysis.
Sampling Recruitment of experimental sample Study or arm ClinicalTrials.gov Key info about how the sample was recruited. Open text — Include if recruitment was random or non-random.
Sampling Sampling from recruited Treatment effect ClinicalTrials.gov How the unit was sampled after recruitment. Text-CV Randomized, Non-randomized, N/A Specify unit of analysis and unit of randomization.
Sampling Deviation from sample inclusion Treatment effect ClinicalTrials.gov Any dropouts prior to treatment assignment. Open text — Used to monitor reporting completeness.
Sampling Baseline average female share Study ClinicalTrials.gov Proportion of females in the study population. Numeric — Control group average if baseline unavailable.
Sampling Baseline average age Study ClinicalTrials.gov Average age of the study population. Numeric — Control group average if baseline unavailable.
6. Intervention
Topic Field Level Relevant Standard Definition Response Options Controlled Vocabulary Notes
Intervention Name of intervention Treatment arm ClinicalTrials.gov Descriptive name used to refer to the intervention arm. Open text — Used for labeling and comparison of arms.
Intervention Details of intervention Treatment arm ClinicalTrials.gov, AEA Registry Details that distinguish the intervention from others. Open text — Includes dosage, duration, method, etc.
Intervention Name of arm Treatment arm ClinicalTrials.gov Label used for each group/arm in the study. Open text — Used for assigning outcome and estimates.
Intervention Number of randomization units per arm Treatment arm — Number of clusters or individuals per arm. Numeric — Used to assess sample size per treatment.
Intervention Mapping of intervention to arms Treatment arm ClinicalTrials.gov How interventions are assigned to each arm. Format — Important for cross-referencing.
Intervention Intervention assignment strategy Study ClinicalTrials.gov Strategy used to assign participants to interventions. Text-CV Simple, complete, stratified, cluster Used to evaluate internal validity.
Intervention Timeline of intervention Treatment arm AEA Registry Start and end date of intervention. Date (YYYY-MM-DD) — Used to align with outcome collection.
7. Outcome
Topic Field Level Relevant Standard Definition Response Options Controlled Vocabulary Notes
Outcome Name of the outcome measure Treatment effect CONSORT Name of the specific outcome being measured. Open text — Should match terminology used in results section.
Outcome Outcome measure category Treatment effect CONSORT General category of outcome (e.g., education, health). Text-CV From predefined list Used for filtering and tagging effects.
Outcome Outcome measurement tool Treatment effect CONSORT The tool or method used to measure the outcome. Open text — Include citations if using standard tools.
Outcome Outcome measure statistical type Treatment effect CONSORT Type of variable: binary, continuous, etc. Text-CV Binary, continuous, categorical, index Verify how outcome is modeled.
Outcome Outcome measure standardized Treatment effect CONSORT Whether the outcome has been standardized. Yes/No/Don't know — Standardization is common in meta-analysis.
Outcome Standardization description Treatment effect CONSORT Description of the standardization process. Open text — Include reference group if z-scored.
Outcome Outcome measure coding Treatment effect CONSORT Whether coding was needed to interpret results. Yes/No/Ambiguous/Don't know — Useful for text-mining approaches.
Outcome Outcome data source Treatment effect CONSORT Where outcome data was obtained. Text-CV Survey, admin data, sensors, etc. Useful for evaluating external validity.
Outcome Outcome measure collection start date Treatment effect CONSORT Start date of outcome measurement. Date (YYYY-MM-DD) — Used to match with intervention timeline.
Outcome Outcome measure collection end date Treatment effect CONSORT End date of outcome measurement. Date (YYYY-MM-DD) — Used to determine follow-up period.
Outcome Outcome mean Treatment effect — Mean value of the outcome for treatment/control. Numeric — Record separately for each group if available.
Outcome Outcome SD Treatment effect — Standard deviation of the outcome. Numeric — Used for calculating standardized effects.
Outcome Outcome sample size Treatment effect — Sample size for the outcome measure. Numeric — Record for both treatment and control.
Outcome Additional information for clustered RCTs Study — ICC, number of clusters, cluster sizes, etc. Open text / Numeric — Important for adjusting SEs.
8. Estimates
Topic Field Level Relevant Standard Definition Response Options Controlled Vocabulary Notes
Estimates Estimation parameter Treatment effect CONSORT Statistical parameter estimated. Text-CV Mean difference, risk ratio, etc. Follow CONSORT reporting standards.
Estimates Estimand Treatment effect CONSORT Target quantity being estimated. Text-CV ATE, ITT, LATE, etc. Clarify population, treatment, outcome, and contrast.
Estimates Null hypothesis Treatment effect CONSORT Hypothesis being tested. Text-CV H0: no effect Used to interpret significance.
Estimates Statistical test of hypothesis Treatment effect CONSORT Statistical test used (t-test, regression, etc.). Text-CV OLS, logit, IV, etc. Critical for replication and comparison.
Estimates Contrast for the treatment effect Treatment effect CONSORT Description of the control group or comparison. Open text — Clarify contrast in multi-arm trials.
Estimates Estimate of the treatment effect Treatment effect CONSORT Numerical value of the effect estimate. Numeric — Report sign, magnitude, and units.
Estimates Unit of measure of the treatment effect Treatment effect — Units in which the estimate is expressed. Open text — Example: test score points, percentage, SDs.
Estimates Precision Treatment effect CONSORT Type of precision measure reported. Text-CV SE, CI, p-value Use standard labels.
Estimates Precision value Treatment effect — Numerical value of the precision measure. Numeric — Ensure it matches the type above.
Estimates Subgroup of heterogeneous analysis Treatment effect — Name or label for the subgroup. Text-CV Gender, age, etc. Indicate subgroup ID if nested in broader group.
Estimates Description of heterogeneous analysis Treatment effect — Description of how subgroup analysis was performed. Open text — Clarify model interaction or stratification.
Estimates Estimates of heterogeneous treatment effects Treatment effect — Effect size for subgroup-specific estimates. Structured response — Multiple rows if needed.
9. Quality and Robustness
Topic Field Level Relevant Standard Definition Response Options Controlled Vocabulary Notes
Quality and Robustness Attrition Study CONSORT Percentage or number of dropouts. Numeric — May be reported for treatment/control separately.
Quality and Robustness Dealing with attrition Study CONSORT How the study dealt with missing observations. Text-CV Listwise deletion, imputation, etc. Indicate robustness checks, if any.
Quality and Robustness Balance test Study CONSORT Whether the study reports a balance test. Yes/No/Not reported/Don't know — Look for baseline covariates comparisons.
Quality and Robustness Compliance Control Study CONSORT Compliance rate in the control group. Numeric — Define clearly; may require calculation.
Quality and Robustness Compliance Treatment Study CONSORT Compliance rate in the treatment group. Numeric — Compare with control for crossover analysis.
Quality and Robustness Covariates included Study CONSORT List of covariates used in the estimation model. Open text — Use to evaluate specification consistency.
Quality and Robustness Imbalanced covariates included Study CONSORT Whether imbalanced covariates were adjusted for. Yes/No/Other — Cross-check with balance test results.
