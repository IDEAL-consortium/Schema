1. Title and Publication Details

This section captures basic bibliographic information about the study, including the full title, citation, and DOI or permanent URL. These details allow coders to link each extracted record to its original source.

8. Estimates

This section captures the effect sizes reported by the authors, along with information about the estimation strategy, statistical tests, precision, and subgroup analyses.

Field Level Relevant Standard Definition Response Options Controlled Vocabulary Notes
Estimation parameter Treatment effect CONSORT – Consolidated Standards of Reporting Trials Statistical parameter estimated. Text-CV Mean difference, risk ratio, etc. Follow CONSORT reporting standards.
Estimand Treatment effect CONSORT – Consolidated Standards of Reporting Trials Target quantity being estimated. Text-CV ATE, ITT, LATE, etc. Clarify population, treatment, outcome, and contrast.
Null hypothesis Treatment effect CONSORT – Consolidated Standards of Reporting Trials Hypothesis being tested. Text-CV H0: no effect Used to interpret significance.
Statistical test of hypothesis Treatment effect CONSORT – Consolidated Standards of Reporting Trials Statistical test used (t-test, regression, etc.). Text-CV OLS, logit, IV, etc. Critical for replication and comparison.
Contrast for the treatment effect Treatment effect CONSORT – Consolidated Standards of Reporting Trials Description of the control group or comparison. Open text — Clarify contrast in multi-arm trials.
Estimate of the treatment effect Treatment effect CONSORT – Consolidated Standards of Reporting Trials Numerical value of the effect estimate. Numeric — Report sign, magnitude, and units.
Unit of measure of the treatment effect Treatment effect — Units in which the estimate is expressed. Open text — Example: test score points, percentage, SDs.
Precision Treatment effect CONSORT – Consolidated Standards of Reporting Trials Type of precision measure reported. Text-CV SE, CI, p-value Use standard labels.
Precision value Treatment effect — Numerical value of the precision measure. Numeric — Ensure it matches the type above.
Subgroup of heterogeneous analysis Treatment effect — Name or label for the subgroup. Text-CV Gender, age, etc. Indicate subgroup ID if nested in broader group.
Description of heterogeneous analysis Treatment effect — Description of how subgroup analysis was performed. Open text — Clarify model interaction or stratification.
Estimates of heterogeneous treatment effects Treatment effect — Effect size for subgroup-specific estimates. Structured response — Multiple rows if needed.

Balance test Study CONSORT – Consolidated Standards of Reporting Trials Whether the study reports a balance test. Yes/No/Not reported/Don't know — Look for baseline covariates comparisons.
